This folder includes all code used in the numerical experiments in Section 
8 for the paper "Robust optimization for models with uncertain SOC and SDP 
constraints" by Jianzhe Zhen, Frans de Ruiter, Ernst Roos & Dick den Hertog.

All the code is authored by Jianzhe Zhen.

--------------------------------------------------------------------------------

The folder contains the following script files with described function:

Robust_Sensor.m
This function computes the prosposed approximation scheme using linear decision rules.

diabete.m
This .m file is obtained the data for the robust regression numerical experiments.
